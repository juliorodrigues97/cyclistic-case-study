# Data Folder

This folder contains the datasets used in the Cyclistic Case Study:

- `Divvy_Trips_2019_Q1.csv`: Trip data for Q1 2019
- `Divvy_Trips_2020_Q1.csv`: Trip data for Q1 2020

Datasets were downloaded from the official Divvy system and anonymized to protect user privacy. Place these files in this folder to run the analysis scripts.
